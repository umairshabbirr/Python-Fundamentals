PROJECT NAME:
AI Personal Finance Tracker (Multi-User System)

PROJECT OVERVIEW:
Ye project Python mein develop kiya gaya hai jo users ko
apni income, expenses aur savings ko manage karne mein help karta hai.
Is system mein multiple users ka support hai aur har user ka data
separate files mein store hota hai.

KEY FEATURES:
1. User Login & Signup System
2. Multiple Users with Separate Data
3. Income Management (Source-wise)
4. Expense Management with Auto Category Detection
5. Academic Fees Category
6. Saving Target Setting
7. Real-Time Saving Alert (Non-blocking)
8. Income & Expense Analysis
9. Reset Income and Reset Expenses Options
10. CSV File-based Data Storage

AI LOGIC USED:
Is project mein rule-based AI logic use ki gayi hai:
- Expense description se category detect hoti hai
- Saving usage par automatic warning generate hoti hai
- Financial rules enforce kiye gaye hain (income ke baghair expense allow nahi)

TECHNOLOGIES USED:
- Programming Language: Python
- IDE: Visual Studio Code
- Data Storage: CSV & Text Files
- Platform: Command Line Interface (CLI)

HOW TO RUN PROJECT:
1. Python 3 install hona chahiye
2. Project folder open karein
3. Terminal open karein
4. Command run karein:
   python main.py
5. Pehle signup karein phir login karein
6. Login ke baad menu se options select karein

PROJECT STRUCTURE:
main.py          -> Main program file
users.txt        -> User login data
data_username.csv -> User specific income & expense data
saving_username.txt -> User specific saving target

FUTURE IMPROVEMENTS:
- Password hashing for better security
- Graphical charts for expense analysis
- Monthly PDF report generation
- Mobile or Web version
